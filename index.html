About Me

I'm Grant Kee, and I love learning new things.

I specialize in full stack web development.

I'm currently developing during the day for Citizen Math!

Where I'm at in my coding journey:

My bread and butter. I enjoy challenging myself with advanced React techniques. I often incorporate Redux with my React projects, but I've also spent a lot of time understanding hooks. I'm currently working on a personal finance app that uses React and Material-UI within a Rails app.
My main focus at work. I'm also working on a personal finance app using RoR, React, and PostgreSQL. RoR is such a powerful tool, I really want to master the basics so I can build CRUD apps in no time.
After the personal finance app is live online, I want to build it out in Java and put it on mobile. I respect Java so much, but I'm stuck on how to incorporate it more into my current projects. After my portfolio is polished a little more and I finish the web version of my current personal finance app, I will be back to Java.
I'm inspired and motivated by Python projects. I currently have a backlog of code that more directly affects my job, but I occassionally have cheat days where I let myself dabble in some Python (like building a basic blockchain to understand the technology better).

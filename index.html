About Me

I'm Grant Kee, and I love learning new things.

I specialize in full stack web development.

I'm currently developing during the day for Citizen Math! In my free time, I'm building a privacy-focused personal finance app.

My Experience

My bread and butter. I enjoy challenging myself with advanced React techniques. I prefer to incorporate Redux with my React projects, but I can pass props from parent to child no problem. I've also spent a lot of time understanding hooks. I'm currently working on a personal finance app that uses React, Material-UI, and PostgreSQL with a Ruby backend.
My main focus at work. People want to say Rails is dead, but I disagree. Rails is a fantastic open-source framework with a vibrant community. If you strap a modern front-end on a Rails app, you're cooking with gas.
I recently completed Andrew Ng's machine learning course through Coursera, and it changed my life. I've decided to keep learning by exploring a sub-categroy of machine learning called Deeplearning. I'm focused on vision applications using TensorFlow using a Jetson Nano.
